Nuevos Métodos

Asignación de propiedad de objeto

  • Nueva función para asignar propiedades enumerables de uno o más objetos fuente a un objeto de destino (método assign)

Búsqueda de un elemento en un arreglo

  • Nueva función para encontrar un elemento en un arreglo find 
    [ 1, 3, 4, 2 ] .find( x => x > 3 ) // 4 
    [ 1, 3, 4, 2 ] .findIndex( x => x > 3 ) // 2

Repetición de cadenas

  • Nueva funcionalidad de repetición de cadenas repeat 
    '' .repeat( 4 * depth ) 
    'foo' .repeat( 3 )

Búsqueda de cadenas

  • Nueva funcionalidad de cadena específica para buscar una subcadena startsWith, endsWith, includes, 
    var cadena = 'hola carola cara de bola' 
     console .log( cadena .startsWith( 'ola', 1 ) ); 
     console .log( cadena .endsWith( 'ola' ) ); 
     console .log( cadena .includes( 'ola' ) ); 
     console .log( cadena .includes( 'ola', 1 ) ); 
     console .log( cadena .includes( 'ola', 2 ) );

Comprobación de tipo de número

  • Nueva funcionalidad para verificar números que no son números y números finitos isFinite 
    Number .isFinite( Infinity ) === false 
    Number .isFinite( -Infinity ) === false 
    Number .isFinite( NaN ) === false 
    Number .isFinite( 123 ) === true

Número de verificación de seguridad

  • Comprobar si un número entero está en el rango seguro, es decir, está representado correctamente por JavaScript (donde todos los números, incluidos los números enteros, son técnicamente números en coma flotante) isSafeInteger

    (+-) 9007199254740991 o (+-) 9,007,199,254,740,991

    Number .isSafeInteger( 42 ) === true 
    Number .isSafeInteger( 9007199254740992 ) === false

Comparación de números

  • Disponibilidad de un valor estándar de Epsilon para una comparación más precisa de los números de coma flotante EPSILON ya que JavaScript no es muy preciso con algunos calculos especialmente con decimales 
    console .log( 0.1 + 0.2 === 0.3 );  // false 
    console .log( Math .abs( ( 0.1 + 0.2 ) - 0.3 ) < Number .EPSILON ); // true

Número de truncamiento

  • Trunca un número de coma flotante a su parte integral, soltando completamente la parte fraccionaria trunc 
    console .log( Math .trunc( 42, 7 ) );  // 42 (Similar a floor()) 
    console .log( Math .trunc( 0.1 ) );  // 0 
    console .log( Math .trunc( -0.1 ) );  // -0

Determinación de signo de número

  • Determine el signo de un número, incluidos los casos especiales de cero firmado y no número sign 
    console .log( Math .sigh( 7 ) ) // 1  
    console .log( Math .sigh( 0 ) ) // 0 
    console .log( Math .sigh( -0 ) ) // -0 
    console .log( Math .sigh( -7 ) ) // -1 
    console .log( Math .sigh( NaN ) ) // NaN
