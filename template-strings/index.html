Maneno de variables de bloque y constantes

Let

  • Indica que una variable sólo va a estar definida en un bloque en particular, al terminar el bloque la variable deja de existir, esto es muy útil para ecitar errores lógicos cuando alteramos una variable que no deberíamos.

Const

  • Previene que una variable declarada cambie de valor, convirtiendose efectivamente en una constante
  • Siempre es recomendable usar constantes para valores que sabemos que no van a cambiar, así se evitan modificaciones
