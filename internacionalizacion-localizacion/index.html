Internacionalización y Localización

  • ES6 tiene un objecto que es el Intl para la internacionalización y localización
  • Este Objeto tiene 3 métodos
    • Collator: Define el tipo de internacionalización de acuerdo al idioma, y permite ordenar de acuerdo al alfabeto de cada país
    • NumberFormat: Permite darle formato a los números y a las monedas de acuerdo a la zona
    • DateTimeFormat: Permite darle formato a las fechas de acuerdo a la zona

    Para ubicar la zona e implementar los métodos anteriores se debe usar un código de lenguaje por zona
