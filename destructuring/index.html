Destructuring

  • Es un nuevo método para extraer datos rápidamente de un Objeto {} o un Arreglo [] sin tener que escribir mucho código
  • Puede ser usado para pasar objetos a una función, permitiendonos obtener propiedades específicas de un objeto
  • Nos permite asignar valores por defecto como argumentos

let foo = [ 'uno', 'dos'. 'tres' ]

let [ one, two. three ] = foo;

console .log( one ); // uno
