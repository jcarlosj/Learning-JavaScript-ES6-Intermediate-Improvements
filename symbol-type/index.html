Symbol Type

  • Son un nuevo tipo de dato primitivo
  • Son únicos e inmutables y son utilizados como identificadores para propiedades de objetos
  • Pueden tener una descripción opcional, pero sólo es utilizada para fines de depurador
  • Cada valor del tipo "Symbol" tiene asociado un valor de tipo "String" o "Undefined" que sirve únicamente como descripción del símbolo
  • El constructor de "Symbol" no debe ser usado con el operador 'new'
  • Tampoco debe ser extendido mediante clases

El siguiente código crea tres símbolos nuevos. Hay que destacar que Symbol( "foo" ) no convierte la cadena "foo" en un símbolo, si no que crea un símbolo nuevo que tiene la misma descripción.

  • var sim1 = Symbol();
  • var sim2 = Symbol( "foo" );
  • var sim3 = Symbol( "foo" );

"foo" es simplemente una descripción para la variable sim2, sim3 de tipo Symbol y funcionan como etiquetas de documentación

Errores comunes

Symbol( "foo" ) === Symbol( "foo" ); // false

var sym = new Symbol(); // TypeError
